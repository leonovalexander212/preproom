# Описание

Дана подсеть IPv4 в формате CIDR, например `10.0.0.0/24`.

Выведи количество доступных адресов хостов: 2^(32 - префикс) - 2 (минус адрес сети и широковещательный). Для /31 и /32 выведи 0.

# Стартовый код

```python
cidr = input().strip()
prefix = int(cidr.split("/")[1])
# выведи число доступных хостов
print(0)
```
