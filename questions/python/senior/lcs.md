# Описание

Найдите длину наибольшей общей подпоследовательности (LCS) двух строк. Подпоследовательность — это последовательность символов, полученная из строки удалением некоторых (возможно, нуля) символов без изменения порядка оставшихся. Первая строка — первая строка s1, вторая строка — вторая строка s2. Выведите длину LCS.

# Стартовый код

```python
def lcs(s1, s2):
    # ваш код
    pass

s1 = input()
s2 = input()
print(lcs(s1, s2))
```
