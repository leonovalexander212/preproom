**1. Удвоение массива**
Реализуй функцию `double(arr)`, возвращающую новый массив, в котором каждое число умножено на 2. Исходный массив не мутировать.

**2. Разворот строки**
Реализуй функцию `reverseString(str)`, которая вернёт строку в обратном порядке. Для пустой строки вернуть `\"\"`.

**3. Максимум в массиве**
Реализуй функцию `max(arr)`, возвращающую наибольшее число. Если массив пустой — вернуть `null`.

**4. Счётчик гласных**
Реализуй функцию `countVowels(str)`, возвращающую количество гласных (a,e,i,o,u) в строке без учёта регистра.

**5. FizzBuzz**
Реализуй функцию `fizzBuzz(n)`, возвращающую массив строк от 1 до n: кратные 3 → \"Fizz\", кратные 5 → \"Buzz\", кратные 15 → \"FizzBuzz\".

### Средние (5)

**6. Уникальные значения**
Реализуй `unique(arr)`, возвращающую массив без дубликатов с сохранением исходного порядка. Решить без `Set` (за O(n²)) и с `Set` (за O(n)) — покажи оба варианта.

**7. Анаграмма**
Реализуй `isAnagram(a, b)`, возвращающую `true`, если строки являются анаграммами друг друга без учёта пробелов и регистра.

**8. Плоский массив 1 уровень**
Реализуй `flatOne(arr)`, разворачивающую массив на один уровень вложенности. Использовать `flat` **нельзя**.

**9. Группировка по свойству**
Реализуй `groupBy(arr, key)`, возвращающую объект `{ [value]: [items] }` для массива объектов.

**10. Capitalize слов**

### Сложные (5)

**11. Глубокое сравнение**
Реализуй `deepEqual(a, b)` для примитивов, массивов и plain-объектов. Учти вложенность и разные типы.

**12. Debounce**
Реализуй `debounce(fn, ms)`, возвращающую функцию, которая вызовет `fn` только через `ms` мс после последнего вызова.

**13. Полифилл Array.prototype.map**
Реализуй `myMap` как метод прототипа массива с сигнатурой стандартного `map(callback, thisArg)`.

**14. Chunk массива**

**15. Sleep через Promise**
Реализуй `sleep(ms)`, возвращающую Promise, который резолвится через `ms` мс. Продемонстрируй использование в `async`-функции с последовательным выводом \"1\", \"2\", \"3\" через 1 секунду между собой.
